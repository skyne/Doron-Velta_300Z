# Doron-Velta

## What's This?

This is a fork & remix of [rogerlz/Doron-Velta](https://github.com/rogerlz/Doron-Velta)

## What's changned?

 - Z got bigger usable space is now ~320mm
 - Motors moved to bottom to lower CoG
